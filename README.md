# enterprise-network-cisco
Cisco Packet Tracer enterprise network for a two-branch business center in Almaty and Taldykorgan. Features floor-based VLANs, inter-VLAN routing, OSPF, RSTP, LACP, GRE over IPsec, ACL, NAT, PPP CHAP/PAP, DHCP, FTP, SSH, and AAA, with focus on scalability, resilience, security, and clear validation.
